# Overview


This tool visualizes player behavior across maps in LILA BLACK using gameplay telemetry data. It enables Level Designers to analyze player movement, combat zones, loot distribution, and survival patterns through an interactive minimap.


#Live Demo


Deployed URL: https://game-nav-helper.lovable.app


#Features
1. Player movement visualization on minimap
2. Event markers (kills, deaths, loot, storm)
3. Filters (map, date, match, event type)
4. Timeline playback for match progression
5. Heatmaps (movement, kills, deaths)
6. Human vs Bot differentiation




This tool helps Level Designers answer:


Where do players move?
Where do fights happen?
Which areas are underutilized?
Are deaths caused by combat or environment?
How does gameplay evolve over time?
